1 Information the Extension Accesses Informasi yang Diakses

The Extension accesses the following data solely to provide its core functionality:

Ekstensi mengakses data berikut semata-mata untuk menjalankan fungsi utamanya:

  • Contact list and phone numbers — entered manually or imported from a local Excel/CSV file by you.
  • Daftar kontak dan nomor telepon — dimasukkan manual atau diimpor dari file Excel/CSV lokal milik Anda.
  • Message template — text you compose inside the extension popup.
  • Template pesan — teks yang Anda susun di dalam popup ekstensi.
  • Media files — images, videos, or documents you choose to attach.
  • File media — gambar, video, atau dokumen yang Anda lampirkan.
  • Delay settings — min/max delay values you configure.
  • Pengaturan jeda — nilai jeda min/maks yang Anda atur.

2 How Data Is Used Bagaimana Data Digunakan

All data above is used exclusively to send WhatsApp messages on your behalf through WhatsApp Web:

Semua data di atas digunakan secara eksklusif untuk mengirim pesan WhatsApp melalui WhatsApp Web atas nama Anda:

  • Contact numbers and variables personalize and address each message.
  • Nomor kontak dan variabel digunakan untuk mempersonalisasi tiap pesan.
  • The message template is rendered and typed into WhatsApp Web's input.
  • Template pesan dirender dan diketikkan ke kolom input WhatsApp Web.
  • Media files are pasted into WhatsApp Web's attachment flow.
  • File media disisipkan ke alur lampiran WhatsApp Web.
  • Settings are saved locally via chrome.storage.local so your session persists.
  • Pengaturan disimpan lokal lewat chrome.storage.local agar sesi Anda tetap tersimpan.

All data is stored locally on your device using Chrome's built-in chrome.storage.local API. No data is ever transmitted to any external server, database, or third party — neither by me nor by the Extension.

Seluruh data disimpan secara lokal di perangkat Anda menggunakan API bawaan Chrome chrome.storage.local. Tidak ada data yang dikirim ke server eksternal, database, atau pihak ketiga manapun — baik oleh saya maupun oleh Ekstensi.

Your contacts and messages never leave your computer.

Kontak dan pesan Anda tidak pernah meninggalkan komputer Anda.
